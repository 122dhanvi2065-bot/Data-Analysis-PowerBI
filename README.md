# Amazon Sales Analysis using Power BI

## 1. Project Overview
Brief introduction about the project and what problem it solves.

## 2. Objectives
- Analyze Amazon sales data
- Identify top performing product categories
- Understand customer ratings and pricing trends
- Create an interactive Power BI dashboard

## 3. Dataset Information
Description of the dataset used in the project.

Example:
The dataset contains:
- Product Name
- Category
- Price
- Rating
- Number of Reviews
- Discount Percentage

## 4. Tools & Technologies Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization

## 5. Data Cleaning & Preparation
Steps performed before analysis:
- Removed missing values
- Formatted columns
- Created calculated columns
- Data transformation using Power Query

## 6. Dashboard Features
Key visuals included in the dashboard:
- Sales by Category
- Top Rated Products
- Price vs Rating Analysis
- Discount Analysis
- Interactive Filters (Slicers)

## 7. Key Insights
Important findings from the analysis.

Example:
- Electronics category has the highest sales
- Products with moderate pricing have higher ratings
- Higher discounts increase product sales
